# HANDOFF — PISMA site

Κατάσταση έργου για συνέχιση σε νέο chat. Τελευταία ενημέρωση: 15/07/2026.

## Το έργο

- **Τι είναι:** Site της PISMA — ψηφιακή παρουσία (site/Google/AI) και αυτοματοποίηση
  διαδικασιών (τιμολόγηση, προσφορές, εισπράξεις) για ελληνικές ΜμΕ. Ιδρυτής: Πάνος, Γλυφάδα.
- **Stack:** Astro (static), καθαρό CSS/TS, χωρίς React/Tailwind. Μόνη dependency frontend: `lenis` (smooth scroll).
- **Repo:** `~/Desktop/pisma` → GitHub `panosbusiness1356/pisma` (branch `main`).
  ΠΡΟΣΟΧΗ: τα `dist/` και `node_modules/` είναι tracked στο git (ιστορικό setup — μην το «διορθώσεις» χωρίς να ρωτήσεις).
- **Deploy:** Vercel → `pisma-flame.vercel.app`, auto-deploy από το `main`.
- **Dev server:** `.claude/launch.json` → `pisma-site` (npm run dev, port 4321).

## Πάγιος κανόνας εργασίας

Μετά από **κάθε** ολοκληρωμένη αλλαγή: `npx astro build` → `git add src/ dist/ (+ ό,τι άλλο άλλαξε)`
→ commit με σύντομο ελληνικό μήνυμα → `git push origin main`. Χωρίς να ρωτάς.
(Υπάρχει και στο memory: always-push-after-changes.)

## Design

- **Θέμα:** σκούρο μωβ-μαύρο (n8n-style), παλέτα/να tokens στο `src/styles/global.css`
  (`--bg #0D0A14`, πορτοκαλί gradient `--grad`, κάρτες `--card/--card-2`).
  Όλα τα χρώματα ΜΟΝΟ μέσω CSS variables.
- **Fonts:** Unbounded (μόνο logo), Commissioner (κείμενο/τίτλοι), JetBrains Mono (`.num`).
- **Smooth scroll:** Lenis στο `Base.astro` (duration 1.15, απενεργοποιείται με prefers-reduced-motion).
- **Nav:** sticky, «μαζεύει» σε κάψουλα με blur στο scroll (`.is-scrolled`), burger ≤1080px,
  links με nowrap. Logo `PISMA.` — ποτέ να μη σπάει η τελεία σε νέα γραμμή.
- **Mobile-first:** το 80% των επισκεπτών είναι από κινητό. Έγινε πέρασμα: κορδέλα 1 γραμμής
  (`.r-short`), σφιχτά sections (44px), βήματα/πακέτα ως συμπαγείς σειρές, footer 2 στήλες,
  κουμπιά full-width στο hero. Κάθε νέο στοιχείο ελέγχεται ΠΡΩΤΑ σε 375px.

## Ήχοι UI

- Web Audio synth στο `Base.astro` (χωρίς αρχεία): «ποπ» στα `.btn`, «τικ» σε links/inputs,
  κουδουνάκι/«ντουπ» στο άνοιγμα/κλείσιμο.
- Διακόπτης στο Nav (`[data-sound-toggle]`, κουμπί 44px με γεμάτο φόντο, εικονίδιο 26px
  με inline διαστάσεις). Προτίμηση στο `localStorage['pisma-sound']`.
- API: `window.pismaSound.{enabled, toggle(), play(name)}`.

## Mascot «Ντοτ» (`src/components/Mascot.astro`)

Η πορτοκαλί τελεία του logo ζωντανή: σύννεφο ~135 πορτοκαλί κουκίδων (seeded στο build),
googly μάτια που ακολουθούν τον κέρσορα, blink, 2 στρώσεις που περιστρέφονται αργά.

- **Γέννηση:** σε κάθε σελίδα από διαφορετικό στοιχείο (map `SPAWNS` στο script — αρχική:
  τελεία logo, Τιμές: `.plan-setup`, Ταμπλό: `.dot`, κ.λπ.). Το στοιχείο πάλλεται, ο Ντοτ
  φουσκώνει με spring. Fallback: `main h1` αν το στοιχείο δεν είναι ορατό.
- **Χορογραφία αρχικής:** μετά τη γέννηση πετάει στο «χωρίς χαμένες ώρες» του h1 και
  ζωγραφίζει τη χειροποίητη υπογράμμιση (`.hl-stroke` SVG, κρύβεται pre-paint με inline
  script, dashoffset συγχρονισμένο με τη θέση του).
- **Ξενάγηση:** στάσεις με `data-mascot-stop` + `data-mascot-say` + `data-mascot-side`
  (+ προαιρετικά `data-mascot-anchor` selector, `data-mascot-pose="sit"`) πάνω στα sections.
  Εναλλάξ πλευρές, συννεφάκι με auto-hide 6,5s, μιλάει 1 φορά ανά στάση, κλικ = replay + boing.
  Σε φαρδιές οθόνες κάθεται στο «λούκι» δίπλα στο περιεχόμενο, σε στενές στη γωνία.
  Δεν πρέπει ΠΟΤΕ να κρύβει μόνιμα κείμενο.

## Περιεχόμενο — ΚΡΙΣΙΜΟ

- **Ύφος:** επαγγελματικό, πληθυντικός ευγενείας («η επιχείρησή σας»), δηλωτικό, χωρίς
  μεταφορές/αστειάκια. Νούμερα παντού (45′, 48 ώρες, 2–4 εβδομάδες, από 590€).
  Λέξεις πελάτη, όχι jargon (όχι «SEO» → «να βγαίνετε στο Google»).
- **Ορολογία:** «Ψηφιακή προβολή», «Αυτοματοποίηση διαδικασιών», «Μεθοδολογία»,
  «Δωρεάν αξιολόγηση 45′» (κύριο CTA, → `/elegxos/`), «Δείτε τις τιμές».
- **Αρχική (`index.astro`):** 8 ενότητες με τη σειρά: Hero → Υπηρεσίες (2 κάρτες με
  παραδοτέα) → Μεθοδολογία (4 στάδια) → Τιμολόγηση 2026 (πορτοκαλί panel) → Αποτελέσματα
  (πριν/μετά, σεναρίως σημασμένα) → Τιμολόγηση (πακέτα δημόσια) → Η εταιρία → Τελικό CTA.
- **Τιμιότητα:** τα «Αποτελέσματα» είναι σεναρίων — πάντα με tag «Ενδεικτικό σενάριο
  εφαρμογής». Μαρτυρίες ΜΟΝΟ επώνυμες όταν υπάρξουν. ΜΗΝ επινοήσεις πελάτες/κριτικές.
- **Κεντρικό config:** `src/config.ts` — ΟΛΑ τα ποσά/ημερομηνίες/στοιχεία από εκεί
  (EINVOICE deadline 1/10/2026, PRICES, ELEGXOS 45′/48h). Email/τηλέφωνο είναι placeholders.

## ΕΚΚΡΕΜΟΤΗΤΕΣ (επόμενα βήματα)

1. **Επαγγελματικό πέρασμα στις εσωτερικές σελίδες** — ΔΕΝ έχει γίνει: se-vriskoun,
   doulevei-mono-tou, times, tablo, elegxos, apodeixeis, epikoinonia, timologisi-2026
   μιλούν ακόμα στον ενικό με το παλιό ύφος. Θέλουν: πληθυντικό, ενιαία ορολογία,
   τίτλους/CTA συνεπείς με την αρχική, και mascot stops με μηνύματα ανά ενότητα.
2. **Εικονίδιο ήχου στο κινητό:** ο χρήστης το έβλεπε «σαν τελεία». Το live HTML
   επιβεβαιωμένα σερβίρει τη διόρθωση (26px inline) — πιθανότατα cache του κινητού του.
   Αν επανέλθει: ζήτα screenshot από το κινητό πριν αλλάξεις κώδικα.
3. **Placeholders:** email `hello@pisma.gr`, τηλέφωνο `+30 210 000 0000` στο config —
   περιμένουν πραγματικά στοιχεία. Πιθανή αγορά domain (peisma.gr/pisma.gr) εκκρεμεί.
4. **Git υγιεινή:** warning για corrupt objects στο `.git/gc.log` (τα push δουλεύουν
   κανονικά). Κάποια στιγμή θέλει καθάρισμα — όχι επείγον.

## Ιστορικό αποφάσεων (για να μην ξανασυζητηθούν)

- Mascot: δοκιμάστηκαν χταπόδι «Οκτώ», μπαλάκια με αξεσουάρ, πυγολαμπίδα/φλόγα/κουβάρι —
  τελική επιλογή ο «Ντοτ» (particle τελεία, στο στυλ cloudstudio.es αλλά πορτοκαλί).
- Ο χρήστης θέλει: κινήσεις αργές/smooth, τίποτα να μην κρύβει κείμενα, εντυπωσιακά
  αλλά απλά πράγματα, και ΠΑΝΤΑ επαγγελματικό τόνο στα κείμενα («όχι μαλακίες»).
- Το «Να σε βρίσκουν πελάτες. Να δουλεύει μόνο του.» έγινε ονόματα υπηρεσιών/σελίδων,
  όχι πια hero τίτλος. Το tagline «Με πείσμα, μέχρι να δουλεύει μόνο του.» μένει στο footer.
