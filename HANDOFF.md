# HANDOFF — PISMA site

Κατάσταση έργου για συνέχιση σε νέο chat. Τελευταία ενημέρωση: 23/07/2026 (ξημερώματα).

## ΝΕΟΣ ΠΑΓΙΟΣ ΚΑΝΟΝΑΣ (23/07/2026): Ο τιμοκατάλογος = πηγή αλήθειας

Το site προβάλλει ΜΟΝΟ υπηρεσίες που υπάρχουν στο `src/data/pricing.ts`. Αφαιρέθηκαν
(απόφαση χρήστη, «τίποτα δεν επιστρέφει»): ηλεκτρονική τιμολόγηση/myDATA/προθεσμία 1/10
(η σελίδα πάρκαρε ως `src/pages/_timologisi-2026.astro` — επαναφορά = μετονομασία χωρίς
underscore + link στο footer), «Τιμολόγια χωρίς πληκτρολόγηση» (καταχώρηση), «Προσφορά
αυθημερόν», «Τέλος οι χαμένες προθεσμίες». Πριν γραφτεί οπουδήποτε νέα υπηρεσία,
έλεγχος ότι υπάρχει στο pricing.ts.

ΠΡΟΣΟΧΗ live checks: το pisma.gr κάνει 308 → **www.pisma.gr** — τα curl πάνε στο www
(με το cookie της κλειδαριάς), αλλιώς τα until-loops κολλάνε.

## Το έργο

- **Τι είναι:** Site της PISMA — ψηφιακή παρουσία (site/Google/AI) και αυτοματοποίηση
  διαδικασιών για ελληνικές ΜμΕ.
- **Stack:** Astro (static), καθαρό CSS/TS, χωρίς React/Tailwind. Μόνη dependency: `lenis`.
- **Repo:** `~/Desktop/pisma` → GitHub `panosbusiness1356/pisma` (branch `main`).
  ΠΡΟΣΟΧΗ: `dist/` και `node_modules/` είναι tracked στο git (μην το «διορθώσεις»).
  Το git identity ορίστηκε τοπικά στο repo (Ariadni Barlou) — αν λείψει, `git config user.name/email` τοπικά.
- **Deploy:** Vercel → **pisma.gr** (www redirect) + pisma-flame.vercel.app, auto-deploy από `main`.
- **ΚΛΕΙΔΑΡΙΑ:** Το site θέλει κωδικό **«pisma2026»** — `middleware.ts` στη ρίζα (Vercel Edge).
  Για curl-έλεγχο στο live: header `Cookie: pisma_code=pisma2026`. Αφαίρεση = σβήσε το αρχείο.
  Όσο υπάρχει, το Google ΔΕΝ βλέπει το site.
- **Dev server:** `.claude/launch.json` → `pisma-site-4322` (port 4322· το 4321 ίσως πιασμένο από άλλο chat).

## Πάγιος κανόνας εργασίας

Μετά από **κάθε** ολοκληρωμένη αλλαγή: `npx astro build` → `git add src/ dist/ (+ ό,τι άλλο)`
→ commit με σύντομο ελληνικό μήνυμα → `git push origin main`. Χωρίς να ρωτάς.
Μετά το push, επιβεβαίωση στο live με curl (με το cookie της κλειδαριάς).

## Design

- **Θέμα:** σκούρο μωβ-μαύρο (n8n-style). Παλέτα ΜΟΝΟ μέσω CSS variables στο `src/styles/global.css`.
  Δοκιμάστηκαν και απορρίφθηκαν: χρωματιστά accents ανά section, φωτεινό θέμα «Κρέμα & Πηλός»,
  χρωματικό «ταξίδι» — ΜΗΝ ξαναπροταθούν (δες memory: color-journey-restore-point).
- **Φόντο:** οι λάμψεις κορυφής ζουν σε `body::before` (στρώμα 100vh) — ΟΧΙ στο body background
  (άφηνε ορατή «γραμμή» στο όριο header/main).
- **Fonts:** Unbounded (logo), Commissioner (κείμενα), JetBrains Mono (`.num`).
- **Nav:** sticky κάψουλα με blur στο scroll. `main { position: relative }` στο global (για τα scatter logos).
  ΠΡΟΣΟΧΗ: το `main.page-in` αφαιρεί την κλάση του μετά το entrance animation (script στο Base) —
  αλλιώς «φυλακίζει» τα position:fixed παιδιά. ΜΗΝ το χαλάσεις.
- **Mobile-first:** ~80% επισκέπτες από κινητό. Κάθε αλλαγή ελέγχεται σε 375/390/430px.

## Αρχική (index.astro)

- Δομή: Hero → Υπηρεσίες → Μεθοδολογία («Πώς δουλεύουμε — σε 4 βήματα») → Τιμολόγηση
  («Οι τιμές μας, δημόσιες — φτιάξτε το πακέτο σας») → Εταιρία → Τελικό CTA.
  Το section Αποτελεσμάτων ΑΦΑΙΡΕΘΗΚΕ από την αρχική (η σελίδα /apodeixeis/ υπάρχει).
- **Hero:** kicker «Ψηφιακή παρουσία & αυτοματοποίηση» (κεντραρισμένο με margin-inline:auto),
  τίτλος με χειροποίητη υπογράμμιση που ζωγραφίζει ο Ντοτ, κουμπιά:
  «Μέτρησε την επιχείρησή σου — 2′» (πορτοκαλί → /tablo/) + «Δωρεάν αξιολόγηση 45′» (ghost → /elegxos/).
- **Κάρτα κριτικών (hero):** λευκή κάρτα με G της Google, ετικέτα «Κριτικές», **4,5** αστέρια
  (πέμπτο μισογεμάτο, .stars-fill width:90%) και μετρητή που ανεβαίνει live έως **(2.000+)**.
  Εκκρεμεί απόφαση: μικρή εκδοχή και σε footer / σελίδα Τιμών.
- **Αιωρούμενα λογότυπα (11, αυθεντικά SVG στο `public/logos/`):** instagram (gradient από Wikimedia),
  facebook, google-gmail, google-icon, whatsapp, google-maps, tiktok (σε λευκό chip), linkedin,
  openai (class inv → invert για λευκό), viber + excel (από Wikimedia — η βιβλιοθήκη gilbarbara δεν τα έχει).
  - Desktop: όλα στο hero (θέσεις εγκεκριμένες — ΜΗΝ αλλάξουν), z-index 2 (πάνω από το mockup ταμπλό).
  - Mobile hero (θέσεις από κύκλους χρήστη): Instagram αριστερά τίτλου (19%), Google δεξιά τίτλου,
    ChatGPT/Excel στο τέλος της παραγράφου (40%).
  - Mobile υπόλοιπα 7: `.sflogo` ΑΓΚΥΡΩΜΕΝΑ ΣΤΑ SECTIONS (όχι % σελίδας — μετατοπιζόταν ανά συσκευή!):
    Gmail δίπλα «ΥΠΗΡΕΣΙΕΣ», WhatsApp τέλος Υπηρεσιών, Maps δίπλα «ΜΕΘΟΔΟΛΟΓΙΑ»,
    LinkedIn πάνω από «ΤΙΜΟΛΟΓΗΣΗ», Facebook δίπλα «ΤΙΜΟΛΟΓΗΣΗ», TikTok πριν «Εταιρία», Viber στο τελικό CTA.
  - Κινήσεις: λίκνισμα (translate/rotate στο img), parallax στο scroll (JS transform στο span, ΜΟΝΟ hero),
    είσοδος «απαλή ανάδυση με ξεθόλωμα» (flogoIn, κλιμακωτά nth-child στο hero, IntersectionObserver
    για τα sflogo). Όλα σέβονται prefers-reduced-motion.
  - Έλεγχος επικαλύψεων: τρέχει προγραμματιστικά (bounding boxes λογότυπα vs h1/h2/h3/p/btn/kicker/li)
    σε 375/390/430 — πρέπει να βγαίνει ΚΑΝΕΝΑ.
- Τελικό CTA: το φωτοστέφανο σβήνει ΜΕΣΑ στο section (radial at 50% 60%) — όχι κομμένη λωρίδα.
